## SDL2:
`yay -S sdl2` </br>
`yay -S sdl2_image` </br>

## Setup
`cd [name]` </br>
`mkdir Build` </br>

## Build and run
`gcc main.c $(sdl2-config --cflags --libs) -o Build/main && ./Build/main` </br>
