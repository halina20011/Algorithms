## SDL2
`yay -S sdl2` </br>
`yay -S sdl2_image` </br>

## Ubuntu
`sudo apt-get install libsdl2-dev` </br>
`sudo apt-get install libpng-dev` </br>

## Setup
`cd [name]` </br>
`mkdir Build` </br>

## Build and run
`gcc main.c $(sdl2-config --cflags --libs) -lm -lpng -o Build/main && ./Build/main` </br>
